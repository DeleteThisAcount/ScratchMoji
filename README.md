# ScratchEmoji
An extension that brings emojis to scratch.

Adds the following emojis (Changes a few scratch made ones):
- ____:)____
- ____:D____
- ____:P____
- ____:|____
- ____thumbsdown____
- ____thumbsup____
- and so much more!

Changes the following text to emojis:
- :)
- :D
- :P
- :|

All emojis are from https://emojiisland.com/pages/free-download-emoji-icons-png
